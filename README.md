# The Official Muku Archive

A playful friendship-themed single-page React website dedicated to Muku.

## Features

- Animated landing page with scrapbook vibes
- Interactive timeline and achievement badges
- Recommendation ignore wall with stamp effects
- Mini-game to keep Muku healthy
- Funny courtroom and Budha profile sections
- Random chatar patar topic generator
- Sticky note memory wall
- Hidden sunflower collectors and final secret message
- Glassmorphism styling and smooth scrolling
- Tailwind CSS + Framer Motion animations
- LocalStorage progress persistence

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
